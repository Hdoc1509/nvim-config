# Keymaps

Used mapleader is `<space>` key.

## File handling

| Mapping     | Mode   | Description                     |
| ----------- | ------ | ------------------------------- |
| `<leader>w` | Normal | Save file                       |
| `<leader>W` | Normal | Save all files                  |
| `<leader>q` | Normal | Save and close file             |
| `<leader>Q` | Normal | Save all files and close Neovim |
